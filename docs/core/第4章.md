## 4. 内在共通ベース

キャラクターの肉体・精神に内在し、かつキャラクターの輪郭を成立させるために最低限必要な骨格情報。全キャラクター共通で使用する。

---

### No.1 種族・身体特徴

人間以外の種族や、特殊な身体的特徴を持つキャラクターのための項目。人間キャラでも超能力や身体的特異性がある場合に使用する。人間で特筆事項がない場合は最低限の記載で構わない。

|項目|内容|項目|内容|項目|内容|
|---|---|---|---|---|---|
|種族の寿命||亜人要素||変身の有無||
|特殊な身体能力||食性||弱点属性||

|項目|説明|記入例|
|---|---|---|
|種族の寿命|その種族の一般的な寿命|約500年 / 不老|
|亜人要素|耳・角・尻尾・翼など人間にない身体パーツ|狐耳と尻尾が3本|
|変身の有無|変身能力の有無とその内容|満月時に狼に変身|
|特殊な身体能力|暗視・飛行・再生能力など|暗闇でも視認可能|
|食性|食事の種類・制限|血液のみ / 雑食だが肉を好む|
|弱点属性|身体的に弱い要素|銀・日光・流水|

### No.3 感覚・嗜好

色・音・季節など感覚的な好みを好き／嫌い／苦手の3軸で記録するカテゴリ。キャラクターの情緒的な深みを表現する。

|項目|好き|嫌い|苦手|
|---|---|---|---|
|色||||
|音楽 / 音||||
|季節||||
|時間帯||||
|天気||||
|場所||||

|項目|説明|記入例（好き → 嫌い → 苦手）|
|---|---|---|
|色|色に対する好み|紺色 → 蛍光色 → ピンク|
|音楽 / 音|音楽ジャンルや音に対する好み|ジャズ → 大音量の電子音 → 金属音|
|季節|季節の好み|秋 → 夏 → 梅雨|
|時間帯|一日の中での好みの時間帯|深夜 → 早朝 → 昼下がり|
|天気|天候の好み|雨 → 猛暑 → 雷|
|場所|場所・空間の好み|図書館 → 遊園地 → 高所|

「嫌い」は積極的に拒否する対象、「苦手」はなんとなくダメ・避けたい対象として使い分ける。

### No.7 ストーリー

キャラクターの物語上の役割と動き。プロットやシナリオ設計の根幹となるパーツ。BONECDは「状態の記録」であるため、ここに記録するのはキャラクターが現在持っている過去・動機・葛藤であり、今後の展開予定ではない。

|項目|内容|項目|内容|項目|内容|
|---|---|---|---|---|---|
|過去||トラウマ||葛藤||
|目的 / 動機||成長の方向性||||
|能力・スキル||弱点||||

|項目|説明|記入例|
|---|---|---|
|過去|物語開始前の重要な経歴・出来事|幼少期に故郷を焼かれ、傭兵団に拾われた|
|トラウマ|心理的な傷・克服すべき過去の経験|仲間を守れなかった記憶|
|葛藤|物語中にキャラクターが抱える内的・外的な対立|復讐と赦しの間で揺れている|
|目的 / 動機|キャラクターが行動する理由・目指すもの|故郷を滅ぼした魔王を倒す|
|成長の方向性|物語を通じてどう変化するか|復讐者から守護者へ|
|能力・スキル|物語上で活用する能力や技術|剣術、火属性魔法、交渉術|
|弱点|物語上で足を引っ張る弱み|単独行動しがち、魔力の暴走|
